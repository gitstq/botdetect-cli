# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-05-15

### Added
- 🎉 Initial release of BotDetect-CLI
- 🔍 30+ browser automation detection signals
- 🌐 Navigator property detection (webdriver, plugins, languages, etc.)
- 🤖 WebDriver detection (Selenium, Puppeteer, Playwright)
- 🔧 Chrome DevTools Protocol detection
- 👆 Browser fingerprint analysis (Canvas, WebGL, Audio, Fonts)
- 🎯 Behavioral pattern detection (Mouse, Keyboard, Scroll)
- 🔐 TLS fingerprint analysis (JA3, JA4)
- 📡 Network signal detection
- 📄 DOM property detection
- 📊 Multiple output formats (JSON, Markdown, HTML, Text)
- 🖥️ TUI dashboard with curses
- 🚀 Zero external dependencies
- 📝 Comprehensive documentation

### Features
- **Detection Modes**: Quick, Standard, Deep, Stealth
- **Browser Support**: Chromium, Firefox, WebKit
- **Report Generation**: JSON, Markdown, HTML, Text
- **TUI Dashboard**: Interactive terminal interface
- **Browser Comparison**: Compare detection scores across browsers

### Documentation
- Multi-language README (English, Simplified Chinese, Traditional Chinese)
- API documentation
- Usage examples
- Contributing guidelines
